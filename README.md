# Hinge-and-Furnace-Model
Computational workflow and processed datasets for the "Hinge-and-Furnace" genomic model. This repository includes scripts for calculating the Azene Mobility Index (AMI) and analyzing IS-proximity and spatially structured selection in bacterial pathogens.
## Installation

### For Python scripts:
Run `pip install -r requirements.txt`

### For R scripts:
Open R and run `source("install_packages.R")`
